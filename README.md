# BEDROCK VIEWER

[![Bedrock Edition](https://img.shields.io/badge/Minecraft-Bedrock-green.svg)](https://minecraft.net)

**BedrockViewer** is a professional-grade platform designed to elevate the Minecraft Bedrock experience. It provides seamless profile searching, and high-fidelity 3D skin rendering, all tailored for the Bedrock ecosystem.

---

## Key Features

### Advanced Visualization
* **3D Skin Viewer: Dynamic skins rendering including Capes and Persona pieces.
* **Capes & Items Explorer:** Detailed inspection of capes and items on Bedrock.
* **Xbox Integration:** Instant access to view the profiles of your Xbox friends.

### Data & Discovery
* **Player Search:** Instant lookup of any player using their unique XUID.
* **NameMC Compatibility:** Full support for NameMC Extras features.
* **Asset Retrieval:** Direct access to original .png files (Feature currently in development).
